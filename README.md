# Pinger

**Pinger** связывает твой телефон с компьютером — уведомления, SMS, звонки, медиа, файлы и фото — по Wi‑Fi, Bluetooth или через облачный relay (E2E‑шифрование).

Это **публичный хаб релизов**. Сборки клиентов автоматически попадают сюда из приватных репозиториев.

## Загрузки

Бери последнюю сборку для своей платформы на странице **[Releases](../../releases)**:

| Платформа | Тег релиза | Файл |
|---|---|---|
| Android | `android-vX.Y.Z` | `.apk` |
| macOS | `macos-vX.Y.Z` | `.app.zip` / `.dmg` |
| Windows | `windows-vX.Y.Z` | `.zip` |

Клиенты сами проверяют обновления (по кнопке и раз в день) и предлагают обновиться.

## Транспорты

Wi‑Fi (mDNS + TCP) → Bluetooth LE → облачный relay. Приоритет — локальная связь; relay включается как fallback.
