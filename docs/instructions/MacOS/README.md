# Pinger для macOS

## О приложении
Клиент Pinger для Mac: показывает уведомления (с возможностью смахнуть их на телефоне), SMS-треды с копированием кодов, журнал звонков тредами, контакты, медиа, фото и файлы с телефона. Обновляется сам (без браузера), умеет запускаться при входе (галочка в меню). Живёт в строке меню (без иконки в Dock). Подключается к телефону по Wi-Fi, Bluetooth или через облако.

## Установка
1. Скачай `Pinger-X.Y.Z.zip` из [Releases](../../../releases) и распакуй.
2. Перетащи **Pinger.app** в «Программы».
3. Первый запуск: правый клик по Pinger.app → **«Открыть»** (приложение подписано ad-hoc, иначе Gatekeeper заблокирует).
4. Кликни иконку Pinger в строке меню → **Show Pairing QR…**. На телефоне нажми «Сканировать QR-код».

## Траблшутинг
- **«Программу не удаётся открыть / повреждена»** — правый клик → «Открыть». Если не помогает: `xattr -dr com.apple.quarantine /Applications/Pinger.app`.
- **Нет окна** — это приложение строки меню; ищи иконку справа вверху, рядом с часами.
- **Не подключается** — телефон и Mac в одной Wi-Fi-сети, либо задай relay. Переспарить: меню → **Clear Pairing**, затем **Show Pairing QR…**.
- **Не находит телефон по Wi-Fi** — убедись, что на телефоне запущен сервис (большая кнопка) и роутер не блокирует mDNS/Bonjour. Помогает **Force Reconnect** в меню.
