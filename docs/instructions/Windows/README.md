# Pinger для Windows

## О приложении
Клиент Pinger для Windows (паритет с macOS): уведомления нативными тостами и списком-зеркалом шторки (✕ смахивает на телефоне), SMS-треды с кликабельными ссылками и копированием кодов (⧉), звонки (входящие — тосты; журнал тредами с историей и кнопкой «Позвонить»), контакты с полными карточками, медиа, фото и файлы (drag & drop). Работает в трее, запускается с системой (управление в Параметры → Автозагрузка), обновляется сам — качает MSIX и ставит без браузера. Подключается по Wi-Fi, Bluetooth или через облако.

## Установка
Два варианта.

**Просто запустить (zip):**
1. Скачай `Pinger-X.Y.Z-win-x64.zip` и распакуй.
2. Запусти `Pinger.App.exe`. (SmartScreen → «Подробнее» → «Выполнить в любом случае».)

**С установкой (MSIX — плитка в Пуске и Start-компаньон):**
1. Скачай `Pinger-X.Y.Z.msix` и `Pinger.cer`.
2. Доверь сертификат один раз (PowerShell от администратора):
   `Import-Certificate -FilePath Pinger.cer -CertStoreLocation Cert:\LocalMachine\TrustedPeople`
3. Установи: `Add-AppxPackage Pinger-X.Y.Z.msix`

Подключение: шестерёнка → QR. На телефоне нажми «Сканировать QR-код».

## Траблшутинг
- **SmartScreen блокирует** — «Подробнее» → «Выполнить в любом случае» (приложение не подписано доверенным сертификатом).
- **MSIX не ставится** — сначала доверь `Pinger.cer` (шаг 2 выше), иначе подпись невалидна.
- **Окно закрылось, но процесс жив** — приложение свернулось в трей. Полный выход: ПКМ по иконке в трее → «Выход».
- **Не подключается** — телефон и ПК в одной Wi-Fi-сети, либо задай relay в настройках (шестерёнка). Переподключиться: трей → «Переподключить».
- **Не запускается (zip)** — нужен Windows 10 1809+ x64; сборка self-contained, рантайм ставить не нужно.
