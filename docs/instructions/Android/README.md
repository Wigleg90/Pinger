# Pinger для Android

## О приложении
Android-приложение — это «телефон» в связке Pinger: оно отдаёт уведомления, SMS, журнал звонков, медиа и файлы на компьютер (macOS/Windows) по Wi-Fi, Bluetooth или через облачный relay (сквозное шифрование). На телефоне крутится фоновый сервис, компьютеры подключаются как клиенты.

## Установка
1. Скачай `Pinger-X.Y.Z.apk` из [Releases](../../../releases).
2. Открой файл и разреши установку из этого источника.
3. Открой Pinger и выдай запрошенные разрешения (уведомления, контакты, SMS, звонки, файлы, Bluetooth).
4. Нажми большую кнопку — сервис запустится.
5. На компьютере открой Pinger и покажи QR. В Android нажми **«Сканировать QR-код»** и наведи камеру.

## Траблшутинг
- **Сервис сам отключается** — отключи оптимизацию батареи: Настройки → Приложения → Pinger → Батарея → «Без ограничений».
- **Не приходят уведомления** — выдай «Доступ к уведомлениям»: Настройки → Уведомления → Доступ к уведомлениям → Pinger.
- **Нет SMS или звонков** — выдай разрешения SMS, «Журнал звонков», «Контакты».
- **Нет файлов/фото** — выдай «Доступ ко всем файлам».
- **Компьютер не подключается** — телефон и ПК должны быть в одной Wi-Fi-сети, либо выбери relay-сервер (кнопка «Выбрать сервер»). После смены ключей переспарь (на ПК «Clear Pairing», затем новый QR).
- **Управление подключениями** — тап по блоку устройства на главном экране показывает активные подключения; оттуда можно удалить пару.
